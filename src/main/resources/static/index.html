API Endpoints

Base URL: http://localhost:8080/api

Authentication:

  • POST /auth/register - Register new user
  • POST /auth/login - Login
  • GET /auth/profile - Get user profile

Songs:

  • GET /songs - Get all songs
  • GET /songs/{id} - Get song by ID
  • GET /songs/search?q={query} - Search songs
  • GET /songs/genre/{genre} - Get songs by genre
  • POST /songs/admin/upload - Upload song with files (admin only)
  • POST /songs/admin - Add song with paths (admin only)
  • PUT /songs/admin/{id} - Update song (admin only)
  • DELETE /songs/admin/{id} - Delete song (admin only)

Playlists:

  • POST /playlists - Create playlist
  • GET /playlists - Get user's playlists
  • GET /playlists/{id} - Get playlist by ID
  • POST /playlists/{id}/songs/{songId} - Add song to playlist
  • DELETE /playlists/{id}/songs/{songId} - Remove song from playlist
  • DELETE /playlists/{id} - Delete playlist

Playback:

  • POST /playback/play/{songId} - Play song
  • POST /playback/pause - Pause playback
  • POST /playback/resume - Resume playback
  • POST /playback/stop - Stop playback
  • GET /playback/current - Get current playback status
